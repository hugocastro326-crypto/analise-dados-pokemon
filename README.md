# analise-dados-pokemon
Análise Exploratória de Dados do Universo Pokémon
Sobre o Projeto

Este projeto realiza uma análise exploratória de dados utilizando Python e a biblioteca Pandas. O objetivo é investigar os atributos dos Pokémon e identificar padrões relacionados à força geral e à classificação entre lendários e não lendários.

Objetivo da Análise

Criar uma métrica consolidada de força chamada "Total"

Identificar os Pokémon mais fortes com base na soma dos atributos

Comparar estatisticamente Pokémon lendários e não lendários

Tecnologias Utilizadas

Python

Pandas

Jupyter Notebook

Principais Etapas

Carregamento do dataset

Criação da variável "Total"

Ordenação dos Pokémon por força

Filtragem de Mega e Primal

Comparação entre lendários e não lendários

Conclusão

A análise demonstrou que Pokémon lendários possuem, em média, maior valor total de atributos. A criação da variável "Total" permitiu uma comparação objetiva da força geral, evidenciando como a análise de dados pode revelar padrões que não são perceptíveis apenas pela observação direta.
